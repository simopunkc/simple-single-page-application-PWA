# Simple Single Page Application (SPA)

## features

- Progressive Web App (PWA)
- Indexed DB Promise
- Materialize
- Workbox

## Build

```sh
npm install
npm run build
```

## Usage

- git clone https://github.com/simopunkc/simple-single-page-application-PWA.git
- "execute index.html in browser"
